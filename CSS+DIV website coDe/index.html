中新网客户端北京6月9日电(冷昊阳)今天，随着新疆、黑龙江等地即将结束最后一日的考试，2019年全国高考也随之落下大幕。在考试结束后，高考成绩何时发布？阅卷要如何保障准确和安全？填报志愿有啥变化？高考结束，这些考后信息需了解。

在山东，该省于8日召开2019年夏季高考第一次新闻发布会，据介绍，该省的答题卡扫描和评卷工作从6月9日开始，到20日结束。评卷期间将安排媒体开放日，向新闻媒体介绍有关评卷工作流程，接受社会监督。

在北京，据媒体报道，该地的阅卷计算机在阅卷前会重装操作系统，剔除大部分不必要的程序软件。阅卷期间所有计算机将组成内部局域网，并与外网实现物理隔离。此外，考生的答卷扫描后将以图片形式存在服务器里。存有答卷数据信息的服务器放置在专用机房，关键评卷场所配备视频监控录像设备，视频监控实现了全方位、无死角。
